# FileRenamer

**FileRenamer** is a Windows application built using **WPF** in **.NET** with **C#**. It allows you to easily rename files, folders, and more within your system through a simple graphical interface.  

## Features
- Rename files and folders efficiently  
- User-friendly WPF interface  
- Works seamlessly on Windows systems with .NET installed  

## Prerequisites
- **.NET SDK** installed on your system  
  You can download it from [Microsoft .NET Downloads](https://dotnet.microsoft.com/download)

## Getting Started
1. Install the .NET SDK if not already installed.  
2. Open a terminal or command prompt.  
3. Navigate to the project folder:  
   ```bash
   cd FileRenamer2
   cd WpfApp1
   dotnet run
4. Project Window should appear and is ready to be tested
